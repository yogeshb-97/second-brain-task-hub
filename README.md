# Second Brain — Daily Task Hub

A gamified personal productivity app with three focused views:
- **Today's Tasks** — what you're doing right now
- **Pending Tasks** — everything on your list
- **Habits** — daily habit tracking with streaks

## Features
- ✅ Add, complete, and delete tasks
- ⚡ XP system — earn points as you finish tasks
- 📈 Level up as you build momentum
- 🔥 Habit streaks and consistency map
- 💾 All data saved locally in the browser
- 📱 Works on iPhone (add to Home Screen for app feel)

## Deploy
This is a single static HTML file — no build step needed.

```bash
npx vercel --prod
```
